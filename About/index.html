I began this project by analysing the application which started the user journey, for example if I wanted to add a product to my shop or and a new page to my website, how did this look and how were the tasks different. I wanted these task for the user to be nearly visually identical so that once the user has done one of them they have done them all. By achieving this the learning curve would be easier to pick up over the application.
