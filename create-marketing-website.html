Create Payments

After launch of Create Payments it needed a landing page, this was to promote the new feature and build awareness. To do this the page is made up from six sections, the following being:

  1. Small intro about Create Payments
  2. Key features
  3. Customer quotes
  4. Insight to Create Payments

From this we added tracking to see how users interacted with the page, to get the best use out of this landing page.
