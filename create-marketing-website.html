Project Overview

Create.net is an online website building platform, giving small and startup businesses the opportunity to build and manage their own ecommerce shop. Below are some of the projects that I have worked on. They include feature pages for new products, the implementation of the re-brand and customer success stories.

Create Website Re-Branded & Re-Built

www.create.net

I worked closely with the company directors to implement Create's new brand identity and the responsive aspect of the website. In order to achieve this I worked with Create's external designer to implement the marketing site designs and the responsive flows of the framework.

Due to the scale of this project I was responsible for the body of the code overhaul. I worked with other team members to delegate sections of the site to meet the deadline.

Other core pages of this update were the Create blog, Features & Pricing page and the HelpCentre.

Create Payments

As part of the launch of Create's new product, Create Payments, I created a landing page to promote the feature and build awareness. The page was made up of six sections:

  1. Small intro about Create Payments
  2. Key features
  3. Customer quotes
  4. Insight to Create Payments
  5. An incentive to join
  6. An action point

Following this, tracking was added to see how users interacted with the page, allowing for future optimisations.
www.create.net/create-payments.phtml

Success Stories
