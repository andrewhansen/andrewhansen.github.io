Project Overview

I was given this project to look at are current template themes how these could be improved, as this was a key feature of creates product. The main goals of this project were:

  • Move away from ‘one design fits all’.
  • A process to create focused design themes.
  • Manage other design team members to work to this process.

Skills & Tools

  • Sketching Logo Designs
  • Photoshop
  • Sketch
  • Affinity Designer

The Process

I began looking at creates current themes at that time, to see what we really offered our users. I printed out all of our templates so I had a whole collection of designs then started grouping them by industries, to see any common patterns were occurring.

So I was looking at different template structures, colour pallets, use of imagery and seeing if there was a connection by industries. Also looking into key design elements that made an industry and what was the majority of users were trying to create. Doing this research really helped to give us a better understanding of creates users and the possible new user we could be attracting.

The process below is how we plan, designed and build the themes. Each stage had a review at the end of it this was to ensure the new theme were going in the right direction and was on time.

Stage 1

  • Pick industry and industry focus, for example 'industry - fashion' and 'industry focus - jewellery'

  • Create a “template-theme” folder, this was to hold all the fonts, colours, graphics, imagery and notes in one place.

  • Research Create users based in the picked industry.

  • Gather / create graphics and imagery for the theme. This was useful for creating mood boards which was a great way to sho off the first stage in an effective visual way.

  • Pick theme name, this was used for the website url name.

Stage 2

  • Design theme logo.

  • Create a style board using sketch, which was made up from the elements below.

  • Adding a logo font, which was the same font as the designed logo and have a fallback font.

  • Heading font, content font and icon font.

  • Colour palette, to show how colour logic would work on background and foreground of the theme.

  • Wallpaper or graphics needed for the user on release.

Stage 3

  • Start buidling the demo site with the use of images, graphics, products and lorem ipsum.

  • Handover to the marketing team to change lorem ipsum.

  • Handover theme assets to the dev team, this was to be added to the application.

  • Start creating screenshots and graphics for the design studio in the application and marketing promotion.
